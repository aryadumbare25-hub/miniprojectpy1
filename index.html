Setup guide

Up and running
in 60 seconds.

No pip installs. No virtual environments. Just Python.

1

Make sure Python is installed

Open your terminal or command prompt and type:

python --version

You need Python 3.6 or higher. Download from python.org if needed.

2

Download the project file

Click the download button below, or clone this repository:

git clone https://github.com/yourusername/finance-tracker
3

Run the program

Navigate to the folder and run:

python finance_tracker.py
4

Follow the on-screen prompts

Enter your name, set a monthly budget, and start tracking! The app walks you through every feature with a simple numbered menu.
